Installing rpcsvc-proto
==================

Installing `rpcsvc-proto` from the main channel can be achieved by:

```
conda install rpcsvc-proto
```

It is possible to list all of the versions of `rpcsvc-proto` available on your platform with `conda`:

```
conda search rpcsvc-proto
```
